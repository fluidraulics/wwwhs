## Setup

- ~~proudly~~ powered by :cactus: [Spike](https://github.com/static-dev/spike), a ~~modern~~ static build tool.
- `npm i spike -g`
- clone this repo and `cd` into it
- run `npm install`
- run `spike watch` or `spike compile`
